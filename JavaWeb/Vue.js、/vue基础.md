# 3. 缩写
## 3.1. v-bind 缩写
Vue.js 为两个最为常用的指令提供了特别的缩写：
![](_v_images/_1541383779_18161.png)

```
<!-- 完整语法 -->
<a v-bind:href="url"></a>
<!-- 缩写 -->
<a :href="url"></a>
```
## 3.2. v-on 缩写
```
<!-- 完整语法 -->
<a v-on:click="doSomething"></a>
<!-- 缩写 -->
<a @click="doSomething"></a>
```
